# Digital Consulting Agüita

Welcome to the official repository for Digital Consulting Aguita, a digital consultancy focused on building scalable web platforms through modern technologies and responsible engineering practices.

## 🎯 Our Vision

We prioritize technical decisions that support sustainable development. Our core mission is to help organizations—especially those dedicated to ocean conservation, marine ecosystems, and environmental impact—create digital solutions that are both technically robust and aligned with long-term ecological goals.

### 🛠 Tech Stack

This project is built with a performance-first approach and strict type safety:

- Framework: Next.js 14 (App Router & MDX Integration)
- Styling: Tailwind CSS v4 for an atomic, lightning-fast design system.
- Content Management: Advanced MDX pipeline (@mdx-js) to transform technical documentation into interactive React components.
- Animations: Framer Motion for fluid, low-overhead UI interactions.
- Type Safety: TypeScript for a reliable and maintainable codebase.
- Code Quality: Prettier (with Tailwind plugin) and ESLint for consistent formatting and best practices.

### 🚀 Key Features

- Responsible Engineering: Optimized code to reduce client-side energy consumption and carbon footprint.
- High-Fidelity MDX: Utilizing shiki for syntax highlighting and rehype for a professional-grade content rendering experience.
- Sustainable Scalability: Built to handle growth without technical debt, using sharp for high-performance image optimization.
- Marine-Focused Design: Specialized layouts tailored for environmental data visualization and storytelling.

### 🧩 Architectural Strengths

This project implements a sophisticated content pipeline:

- Unified Ecosystem: We use a chain of plugins (```remark-gfm```, ```remark-unwrap-images```) to ensure that scientific data and environmental reports are rendered with precision.
- Efficiency by Design: Using ```recma-import-images``` and ```unified-conditional```, we ensure that only the necessary assets are delivered to the user, adhering to our "low-carbon web" philosophy.
